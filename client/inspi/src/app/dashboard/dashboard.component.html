¡Bienvenido!

Esta es la página de creación de evaluaciones para evaluación de calidad del INSPI.

En esta página podrá crear plantillas, las cuales serán usadas como referencias para la creación de una campaña. Las campañas van a ser las evaluaciones que se les tomará a los diferentes laboratorios. Aquí podrá crear viales que tendrá la respuesta la cual en un futuro podrá ser referenciado a la evaluación que haríacada laboratorio
¡Esperamos que esta plataforma les sea de mucha utilidad!
